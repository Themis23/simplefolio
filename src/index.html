Full Stack React.js Image Recognition App

This project is a dynamic full-stack application combining React.js, Express.js, PostgreSQL, and the Clarifai API. The front end is developed using React.js and features user authentication with login and registration forms, ensuring secure storage of user data through bcrypt password hashing. On the server side, Express.js handles routing and interacts with a PostgreSQL database to store user information. When users log in, they can submit image URLs containing faces. Leveraging the Clarifai API's image recognition capabilities, the application detects faces in the images and updates the user's entry count for the pictures added. This project showcases the integration of front-end and back-end technologies to create an efficient and intelligent image processing application.
